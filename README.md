# 个人主页

这是一个简约优雅的个人主页项目，采用纯静态页面设计，无需后端支持。

## 功能特点

- 🎨 动态背景切换
  - 自动轮播多张精美风景图片
  - 三种不同的视觉效果（模糊、高清、毛玻璃）
  - 平滑的过渡动画

- ⏰ 实时时钟显示
  - 显示格式：YYYY-MM-DD HH:mm:ss
  - 自动更新

- 📱 响应式设计
  - 适配各种屏幕尺寸
  - 移动端友好

- 🎯 页面功能
  - 首页：展示个人信息和社交媒体链接
  - 关于我：展示个性签名和名言
  - 文章列表：预留的文章展示区域

## 技术栈

- HTML5
- CSS3
- JavaScript (原生)
